# 2.3 保留文字格式

`<pre>...</pre>`：**pre**formatted text。

## 語意

保留原來的文字格式。會保留多個空格、斷行。

## 範例

留意：若有縮排，也會被呈現。

```markup
<pre>
這個文字    左側有多個空格
也可以斷行
</pre>
```
