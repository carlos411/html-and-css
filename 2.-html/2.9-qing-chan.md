# 2.15 清單

`<dl>`：**d**efinition **l**ist‌

`<dt>`：**d**efine **t**erm‌

`<dd>`：**d**efinition **d**escription‌

### 範例

當有多個名詞想要去做說明時，就會用 `<dl>` 的結構：

```markup
<dl>
  <dt>www</dt>
  <dd>是 World Wide Web 的縮寫</dd>
  <dt>拿鐵咖啡</dt>
  <dd>加了牛奶的咖啡</dd>
</dl>
```
