# 2.17 獨立內容

`<figure>...</figure>`：表示是一段獨立的內容。

`<figcaption>...</figcaption>`：上述獨立內容的標題。放在 \<figure> 標籤中。

## 範例

```markup
<figure>
  <img src="http://notes.carlos-studio.com/images/butterfly.png" alt="蝴蝶" title="蝴蝶">
  <figcaption>這是蝴蝶的照片，此描述會視為是圖片的標題</figcaption>
</figure>
```
