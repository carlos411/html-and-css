# 3.24 表單樣式

## textarea 多行文字框

HTML

```markup
<textarea>這是內容</textarea>
```

CSS

```css
textarea{
  resize: both;
}
```

結果如下：

![](../.gitbook/assets/textarea\_resize\_sample.png)

resize 可選的值有：

* `both`：這是預設。使用者可改變 textarea 欄位的寬高。
* `horizontal`：使用者可水平將 textarea 欄位變寬。
* `vertical`：使用者可垂直將 textarea 欄位變高。
* `none`：使用者無法改變 textarea 欄位的寬高。
