# 3.25 關於 outline

## 說明

outline\(輪廓\) 是一個位在 border 邊框\(border\)外面的線，可以設定粗細、線的類型、顏色、位移。outline 並不會影響周圍的元素。

註：outline 並不能針對某一邊去設定。

## 語法

```css
outline: 1px solid red; /* 預設是 none */
outline-offset: 5px;    /* 設定向內或向外擴展，也可以是 -5px，就會向內縮 */
```
